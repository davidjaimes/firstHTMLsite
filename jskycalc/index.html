JSkyCalc on macOS Sierra




Step-by-step instructions for install and setup.




Step 1: Go to Oracle-Java website and download thedk-8u121-macosx-x64.dmg file.


Step 2: Go to JSkyCalc website and download the executable tar file.


Step 3: Use command mkdir ~/JSkyCalc to create an empty directory in your home directory and name it JSkyCalc. Then, move executable tar file JSkyCalc.jar into this directory.


Step 4: Use command jar -xvf JSkyCalc.jar from within ~/JSkyCalc directory to unpack jar file.


Step 5: Edit skycalcsites.dat to add Mount Laguna Observatory as the first site (above Kitt Peak)--it will become the default. You may have to comment out an observatory to make room. 




"Mount Laguna Obs.", 7.76173, 32.84, 8., 1, "Pacific", "P", 1859., 1859.



Step 6: Add following lines to your ~/.bash_profile file. 




# use jsky command in any directory to start JSkyCalc
alias jsky='java JSkyCalc'
CLASSPATH=$CLASSPATH:~/JSkyCalc
export CLASSPATH



Step 7: Finally, use the command source ~/.bash_profile to source the file and then use jsky command to start using JSkyCalc from any directory!


Step 8: The following line is the general format for JSkyCalc (.dat) files. 




Name_no_blanks hh mm ss dd mm ss equinox +++



It's fine if all your stars have equinox=2000. Here is an example of observe-star.dat formatted for JSkyCalc. 




some_star 00 37 52.92 -09 42 16.5 2000.0 +++
