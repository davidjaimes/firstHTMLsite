Terminal.app on macOS

Customizing the Terminal.app bash prompt with color.


Figure 1: Custom colored Terminal.app for macOS.


The following code is my .bash_profile file. 


 1 # Custom colors for bash Terminal in macOS
 2 # Default is:  “exfxcxdxbxegedabagacad”
 3 export LSCOLORS=gxgxcxdxbxegedabagacad
 4 export CLICOLOR=1
 5 # Custom colored bash prompt
 6 C0="\e[38;5;120m\]"  # Light Green
 7 C1="\e[38;5;227m\]"  # Light Yellow
 8 export PS1="\[$C0\u.com [\[\e[0m\]\[$C1\W\[\e[0m\]\[$C0] $ \[\e[0m\]"
 9 # added from https://www.dartmouth.edu/~physics/labs/skycalc/flyer.html
10 export CLASSPATH=$CLASSPATH:~/JSkyCalc
11 # added by Miniconda3 installer
12 export PATH="/Users/djaimes/miniconda3/bin:$PATH"
13 # Collection of aliases for djaimes.com bash commands
14 alias jsky="java JSkyCalc"
15 alias convert="ipython nbconvert --to latex"
16 alias jynb="jupyter notebook"
17 alias jlab="jupyter lab"
18 alias coffee="caffeinate -u -t"
19 alias ls="ls"
20 alias stat="git status"
21 alias add="git add"
22 alias com="git commit -m"

Part 1: Adding color to your Terminal


Add following lines to .bash_profile file.



# Custom colors for bash Terminal in macOS Sierra (Unix)
# Default is:  “exfxcxdxbxegedabagacad”
export LSCOLORS=bxgxcxdxbxegedabagacad
export CLICOLOR=1


For more information on letter meaning check out the OSX Daily website. You can also look through this site for more explanations go to the Norbauer website.



Part 2: Adding color to bash prompt



Add following lines to .bash_profile file.



# Custom bash prompt via kirsle.net/wizards/ps1.html
export PS1="\[$(tput setaf 6)\]david jaimes bash shell \[$(tput setaf 1)\][\W] \[$(tput setaf 6)\]$ \[$(tput sgr0)\]"


For more information on custom prompt generators go to Noah Petherbridge's website titled Kirsle.
