Custom domain name with Google Domains, Github, and Cloudflare




I already had a personal webpage from Github Pages but I wanted to move away from the default username.github.io convention and register my own custom domain. I found you can purchase a custom domain name from Google if you already have a Gmail account. Just go to Google Domains, search for a name you like and purchase it. It is that easy!

To setup my custom domain name from GitHub, I went to the "Settings" tab in my username.github.io repository and added it. After a few failed attempts to get the page up and running, I discovered from the help pages I needed to add either an ALIAS or ANAME record. However, since Domains does not support either ALIAS or ANAME records, I followed the directions (Github A record settings) to setup an "A" record instead. Put both IPv4 addresses at the same time by using the plus sign. Change from davidjaimes.github.io to davidjaimes.run.

Another nice feature from Google Domains is the ability to add up to 100 email aliases. Instead of giving out a Gmail address--a little unprofessional in my case--I added a few email aliases. Now I can give out astro@davidjaimes.run instead of username@gmail.com. Pretty cool!

So, there you have it. If you're like me and already have a GitHub webpage you'd like to switch to a personal custom domain, follow these easy steps to get your site up and running. Send me an email and let me know if you have any questions or suggestions. Thank you!
